# HR Birthday Management

This module adds birthday management to Employees settings.

## Features

- Enable or disable birthday management from Employees settings
- Send a monthly birthday reminder to the configured HR manager
- Send automatic birthday greeting emails to employees on their birthday
- Create an HR activity when the employee has no work email

## Configuration

Go to `Employees -> Configuration -> Settings` and configure:

- `Enable Birthday Management`
- `Birthday Reminder Manager`
- `Send Monthly Birthday Reminder`
- `Send Automatic Birthday Greeting to Employees`
- `Birthday Greeting Email Template`
